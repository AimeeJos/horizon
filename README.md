## Run Locally

Clone the project

```bash
  git clone https://link-to-project
  cd my-project
```

Build docker image.

```bash
  docker compose build
```

Start the server

```bash
  docker compose up
```
Run the server, api documentaion:

```bash
  https://localhost/api/swagger
```
